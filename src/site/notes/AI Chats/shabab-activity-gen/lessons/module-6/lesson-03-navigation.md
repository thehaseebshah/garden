# Navigation

## Learning Objectives

- Spatial awareness, route planning, observation, direction reading, teamwork, memory, communication, quick decision-making, leadership, patience, and practical confidence in unfamiliar spaces.

## Activity 1 (Landmark Relay)

### Data

| Field | Details |
| --- | --- |
| Age Limit | 10+ |
| Group Size | 4-10 players per team |
| Duration | 20-25 minutes |
| Prep Time | 5-10 minutes |
| Difficulty | Easy to Medium |
| Category | Outdoor, Navigation, Team strategy, Movement |
| Items Needed | 4-6 simple landmarks, paper slips, markers or cones |
| Murabbi Rating |  |

### Setup

Choose a simple open area and place 4 to 6 landmarks. Use natural or easy objects such as a tree, bench, cone, wall, door, or water bottle station.

Use these route examples:

- Start at the gate, walk to the tree, then to the bench, then back to the gate
- Start at the cone, move to the wall, then to the bottle station, then finish at the start point
- Start at the door, pass the chair, then stop at the shadow spot, then return

### How to Play

1. Divide shabab into teams.
2. Show one route to the first runner for 10 seconds.
3. Hide the route and let the runner repeat it from memory.
4. The runner returns to the team and tags the next person.
5. The next person repeats the same route.
6. Teams get points for accuracy, speed, and teamwork.

### Murabbi Notes

Keep the route short and visible. If the group is younger, reduce the route to 3 steps. The goal is direction memory and calm movement, not athletic pressure.

## Activity 2 (Build the Map)

### Data

| Field | Details |
| --- | --- |
| Age Limit | 10+ |
| Group Size | 4-8 players per team |
| Duration | 20-30 minutes |
| Prep Time | 5 minutes |
| Difficulty | Easy to Medium |
| Category | Indoor or Outdoor, Map reading, Creativity, Teamwork |
| Items Needed | Chalk or tape, paper, pen, optional sticky notes |
| Murabbi Rating |  |

### Setup

Draw a simple map of the space on the floor, board, or paper. Mark a start point, 3 landmarks, and a finish point.

Use these map examples:

- Masjid entrance to wudu area to prayer line to exit
- Home gate to water point to tree to shelter
- School door to class area to bench to assembly point

### How to Play

1. Give each team the same rough layout.
2. Teams must draw a route using arrows or numbered steps.
3. One teammate explains the route to the murabbi.
4. Another teammate walks the route on the floor.
5. Points are awarded for clear direction and correct order.

### Creative Challenge

Ask teams to rename the landmarks using simple code words, such as `base`, `bridge`, `shade`, or `safe point`, then explain the route again.

### Sample Answers for Murabbi

- A good route should have a clear start, clear turns, and a clear finish.
- A strong team uses names, numbers, and short instructions.
- If someone gets confused, the team should pause and re-explain instead of rushing.

### Murabbi Notes

Keep the energy playful but disciplined. This lesson should train shabab to notice where they are, describe where they are going, and guide others with confidence.
