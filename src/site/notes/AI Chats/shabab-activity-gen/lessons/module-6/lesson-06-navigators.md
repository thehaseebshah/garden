# Navigators

## Learning Objectives

- Leadership, spatial awareness, clear instruction, active listening, trust, teamwork, route memory, calm correction, responsibility, and guiding others with confidence.

## Activity 1 (Guide the Team)

### Data

| Field | Details |
| --- | --- |
| Age Limit | 10+ |
| Group Size | 4-8 players per team |
| Duration | 20-25 minutes |
| Prep Time | 5 minutes |
| Difficulty | Easy to Medium |
| Category | Outdoor, Leadership, Navigation, Team strategy |
| Items Needed | Cones, chairs, tape, blindfolds or eye covers |
| Murabbi Rating |  |

### Setup

Create a short safe path with 4 to 5 turning points. One person is the navigator and the rest are the team.

Use these route examples:

- Start, turn left, pass the chair, stop at the cone, return
- Start, move straight, step around the marker, turn right, finish
- Start, walk to the wall, turn back, stop at the bench, end

### How to Play

1. Pick one navigator for each round.
2. The rest of the team stands behind the start line.
3. The navigator gives short directions only: left, right, stop, go, turn back.
4. The team follows the path together.
5. If the team gets lost, the navigator must correct them calmly.
6. Switch the navigator after each round.

### Creative Twist

Before the round starts, the navigator must name each landmark with a code name such as `base`, `bridge`, `shade`, or `finish`.

### Sample Answers for Murabbi

- A good navigator speaks clearly and uses short commands.
- A good team listens first and moves second.
- If the path is unclear, the navigator should pause and re-explain rather than shout.

### Murabbi Notes

Keep the path simple and safe. This lesson is about guiding others well, not speed alone.

## Activity 2 (Landmark Captain)

### Data

| Field | Details |
| --- | --- |
| Age Limit | 10+ |
| Group Size | 4-10 players per team |
| Duration | 15-20 minutes |
| Prep Time | 5 minutes |
| Difficulty | Easy |
| Category | Indoor or Outdoor, Memory game, Leadership, Communication |
| Items Needed | 3-5 objects or markers, paper, pen |
| Murabbi Rating |  |

### Setup

Place 3 to 5 simple landmarks in a small area.

Use these landmarks:

- Tree
- Chair
- Cone
- Wall
- Water bottle

### How to Play

1. One player becomes the captain.
2. The murabbi gives the captain a route using the landmarks.
3. The captain must repeat the route to the team in the correct order.
4. The team then walks the route.
5. The captain earns points if the team arrives correctly.

### Sample Route Prompts

- Go to the tree, then the cone, then the wall, then return
- Start at the chair, move to the water bottle, then the tree, then stop
- Go from the wall to the cone, then back to the chair

### Murabbi Notes

Rotate captains quickly so everyone gets a turn. This builds leadership and trust without making the game complicated.
