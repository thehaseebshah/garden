# Emergency Signaling

## Learning Objectives

- Clear communication, signal recognition, teamwork, quick thinking, calm under pressure, pattern memory, problem-solving, leadership, and practical rescue awareness.

## Activity 1 (Signal Towers)

### Data

| Field | Details |
| --- | --- |
| Age Limit | 10+ |
| Group Size | 4-10 players per team |
| Duration | 20-25 minutes |
| Prep Time | 5 minutes |
| Difficulty | Easy to Medium |
| Category | Outdoor, Communication, Team strategy, Movement |
| Items Needed | Cones, sticks, cloth, whistle, stopwatch |
| Murabbi Rating |  |

### Setup

Mark 3 signal stations in an open space. At each station, place a simple object such as a cone, cloth, stick, or chair.

Use these signal rules:

- One whistle = stop and listen
- Two whistles = move to the next station
- Cloth raised high = team is ready
- Arm waving = urgent help needed

### How to Play

1. Divide shabab into teams.
2. The murabbi gives a signal pattern.
3. Teams must move to the correct station and perform the right signal.
4. Example pattern: whistle, whistle, cloth up, move to station 2, wave arm.
5. Teams score for correct order and calm response.

### Sample Signal Patterns

- whistle, stop, cloth up, move right
- whistle, whistle, crouch, move to station 3
- wave arm, hold position, whistle, return to start
- cloth up, circle once, stop, call for help

### Murabbi Notes

Keep the signals simple and repeat them once before the round starts. The goal is fast recognition, not confusion.

## Activity 2 (Rescue Code Relay)

### Data

| Field | Details |
| --- | --- |
| Age Limit | 10+ |
| Group Size | 4-8 players per team |
| Duration | 15-20 minutes |
| Prep Time | 5 minutes |
| Difficulty | Easy |
| Category | Indoor or Outdoor, Memory game, Communication, Teamwork |
| Items Needed | Paper, pens, optional whistle or flashlight |
| Murabbi Rating |  |

### Setup

Write 3 short rescue messages on paper or say them aloud.

Use these examples:

- We are here, please help
- Group is safe, adult needed
- One person needs assistance
- Stay together at the safe point

### How to Play

1. Divide shabab into teams in a line.
2. Show the first player a message for 5 seconds.
3. The player passes the message to the next person by whispering or acting it out.
4. The final player says the message aloud or writes it down.
5. Compare the original message with the final message.

### Creative Challenge

Make the message into a signal form instead of normal speech:

- 1 whistle = help
- 2 whistles = stop
- raised hand = stay there
- flashing torch = attention

### Sample Answers for Murabbi

- Good signals are short, repeatable, and easy to see or hear.
- Good rescue communication tells others where you are, whether you are safe, and whether help is needed.
- A weak signal is unclear, too long, or easy to confuse.

### Murabbi Notes

This lesson should feel active and fun. Focus on training shabab to send a clear message quickly, not on making complicated code.
