# First Principles

## Learning Objectives

- Breaking problems into basic truths, separating assumptions from facts, simplifying complex ideas, thinking creatively, and rebuilding stronger solutions.

## Activity 1 (Strip It Down)

### Data

| Field | Details |
| --- | --- |
| Age Limit | 10+ |
| Group Size | 4-8 players per team |
| Duration | 20-25 minutes |
| Prep Time | 5 minutes |
| Difficulty | Easy to Medium |
| Category | Indoor, Critical thinking, Problem-solving, Teamwork |
| Items Needed | Whiteboard or paper, pens, simple problem prompts |
| Murabbi Rating |  |

### Setup

Choose a simple problem and write it on the board.

Use these problem prompts:

- Our team keeps wasting time before starting
- The group cannot agree on a plan
- A task feels too hard to begin
- We need a better way to organize materials
- We want to improve attendance at the activity

### How to Play

1. Divide shabab into teams.
2. Give one problem prompt to each team.
3. Teams answer 3 questions: What is the real problem? What is definitely true? What is only assumed?
4. Teams reduce the problem into 2 or 3 basic truths.
5. Each team shares its stripped-down version.

### Sample Answers for Murabbi

- The real problem is not laziness alone; the group may not have a clear starting step.
- The real problem is not disagreement alone; the group may lack a shared goal.
- The real problem is not difficulty alone; the task may need smaller steps.
- The real problem is not bad attendance alone; the schedule or reminder system may be weak.

### Murabbi Notes

Push the teams to remove extra words and focus on what is actually true. This helps them stop guessing too early.

## Activity 2 (Rebuild the Solution)

### Data

| Field | Details |
| --- | --- |
| Age Limit | 10+ |
| Group Size | 4-8 players per team |
| Duration | 20-25 minutes |
| Prep Time | 5 minutes |
| Difficulty | Easy |
| Category | Indoor, Innovation, Strategy, Communication |
| Items Needed | Paper, pens, whiteboard |
| Murabbi Rating |  |

### Setup

Take the stripped-down problem and ask the team to build a new solution from the bottom up.

Use these rebuild prompts:

- Start from the smallest useful step
- Remove anything unnecessary
- Add only what solves the core problem
- Keep the plan simple enough to repeat

### How to Play

1. Give each team one stripped-down problem.
2. Teams create a solution using only the core facts.
3. They must present it in 3 steps or fewer.
4. Murabbi checks whether the solution actually addresses the root issue.

### Creative Challenge

Ask teams to improve the plan by removing one extra step and explaining why it is not needed.

### Sample Answers for Murabbi

- If the issue is unclear starting steps, the solution may be: write the first action, assign a leader, and begin.
- If the issue is disagreement, the solution may be: define the goal, list 2 options, and choose one by shura.
- If the issue is a hard task, the solution may be: split it into small parts, assign roles, and begin with the easiest part.
- If the issue is bad attendance, the solution may be: send reminders early, set a clear time, and make the lesson more welcoming.

### Murabbi Notes

Do not let the teams overcomplicate the answer. First principles should make things simpler, not more confusing.
