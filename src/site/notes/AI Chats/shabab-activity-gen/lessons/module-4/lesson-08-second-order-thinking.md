# Second-Order Thinking

## Learning Objectives

- Thinking beyond the first result, comparing short-term and long-term outcomes, making wiser choices, and understanding tradeoffs and consequences.

## Activity 1 (First Effect, Second Effect)

### Data

| Field | Details |
| --- | --- |
| Age Limit | 10+ |
| Group Size | 4-10 players per team |
| Duration | 20-25 minutes |
| Prep Time | 5 minutes |
| Difficulty | Easy to Medium |
| Category | Indoor, Critical thinking, Discussion, Strategy |
| Items Needed | Whiteboard or paper, pens |
| Murabbi Rating |  |

### Setup

Write a decision on the board and ask the team to trace the first effect and the second effect.

Use these decision examples:

- Skip planning and start immediately
- Spend all the budget on one item
- Choose the easiest task first every time
- Share information without checking it
- Ignore a small problem because it seems harmless

### How to Play

1. Divide shabab into teams.
2. Give one decision example to each team.
3. Teams write the first effect and the second effect.
4. They explain whether the decision is wise or risky.
5. Murabbi awards points for clear consequence chains.

### Sample Answers for Murabbi

- Skip planning and start immediately: first effect = start faster; second effect = confusion, mistakes, and wasted time.
- Spend all the budget on one item: first effect = one strong purchase; second effect = no resources left for other needs.
- Choose the easiest task first every time: first effect = quick progress; second effect = hard tasks pile up.
- Share information without checking it: first effect = fast sharing; second effect = spread of false or harmful claims.
- Ignore a small problem because it seems harmless: first effect = less trouble now; second effect = the problem grows.

### Murabbi Notes

Keep the chain short and real. The lesson is to pause before acting and ask what comes next.

## Activity 2 (Consequence Map)

### Data

| Field | Details |
| --- | --- |
| Age Limit | 10+ |
| Group Size | 4-8 players per team |
| Duration | 20-25 minutes |
| Prep Time | 5 minutes |
| Difficulty | Easy |
| Category | Indoor, Analysis, Communication, Problem-solving |
| Items Needed | Paper, pens, arrows or board space |
| Murabbi Rating |  |

### Setup

Create a simple map with one decision in the middle and two or three possible outcomes.

Use these consequence prompts:

- If we rush, what happens next?
- If we wait, what happens next?
- If we choose the short-term gain, what is the long-term cost?

### How to Play

1. Give each team one decision.
2. Teams draw arrows showing the first effect and the second effect.
3. They add one possible benefit and one possible cost.
4. One player explains the map.
5. Murabbi checks whether the team thought beyond the immediate result.

### Creative Challenge

Ask teams to add one extra arrow showing a hidden consequence that might appear later.

### Sample Answers for Murabbi

- Rushing may feel fast, but it can create rework later.
- Waiting may feel slow, but it can prevent mistakes.
- Short-term gain may reduce flexibility later.
- A small problem ignored today can become a bigger problem tomorrow.

### Murabbi Notes

This lesson should make shabab slow down and think ahead. The best answer is the one that sees farther, not just faster.
