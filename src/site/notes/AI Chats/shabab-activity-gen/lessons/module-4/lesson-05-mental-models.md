# Mental Models

## Learning Objectives

- Using structured frameworks, matching the right model to the problem, simplifying decisions, improving judgment, and thinking in clear steps.

## Activity 1 (Model Match)

### Data

| Field | Details |
| --- | --- |
| Age Limit | 10+ |
| Group Size | 4-8 players per team |
| Duration | 20-25 minutes |
| Prep Time | 5 minutes |
| Difficulty | Easy to Medium |
| Category | Indoor, Critical thinking, Strategy, Teamwork |
| Items Needed | Model cards, paper, pens |
| Murabbi Rating |  |

### Setup

Prepare a few simple mental models and a set of problems.

Use these model examples:

- Root cause: find the real source of the problem
- Cost vs benefit: compare what is gained and what is lost
- 80/20 rule: focus on the few things that matter most
- Second order thinking: consider the next result, not only the first result
- Work backward: start from the goal and reverse the steps

Use these problem examples:

- The team keeps arriving late
- The task takes too long to finish
- The group makes mistakes at the same point every time
- A choice looks easy but may create bigger problems later
- We need the fastest way to improve one area first

### How to Play

1. Divide shabab into teams.
2. Show one problem and 3 possible models.
3. Teams choose the best model and explain why.
4. Murabbi asks them to give one practical example.
5. Switch problems and repeat.

### Sample Answers for Murabbi

- Arriving late: work backward or root cause.
- Task takes too long: root cause or 80/20.
- Same mistake every time: root cause.
- Easy choice with future problems: second order thinking.
- Fastest way to improve one area: 80/20.

### Murabbi Notes

The point is not memorizing names alone. The point is choosing a useful lens for the situation.

## Activity 2 (One Problem, Three Lenses)

### Data

| Field | Details |
| --- | --- |
| Age Limit | 10+ |
| Group Size | 4-10 players per team |
| Duration | 20-25 minutes |
| Prep Time | 5 minutes |
| Difficulty | Easy |
| Category | Indoor, Analysis, Communication, Problem-solving |
| Items Needed | Whiteboard or paper, pens |
| Murabbi Rating |  |

### Setup

Choose one real-life problem and solve it using 3 different models.

Use these problem prompts:

- Our group is not organized
- We have too many tasks and not enough time
- A plan works now but causes confusion later

### How to Play

1. Give each team one problem.
2. Teams must solve it using 3 lenses: root cause, cost vs benefit, and second order thinking.
3. They write one sentence for each lens.
4. One speaker presents the three answers.
5. Murabbi compares which lens gave the clearest insight.

### Creative Challenge

Ask teams to invent their own simple model name for the problem and explain how it works.

### Sample Answers for Murabbi

- Our group is not organized: root cause = unclear roles; cost vs benefit = more planning time now saves time later; second order = a messy start creates more confusion later.
- Too many tasks and not enough time: 80/20 = focus on the most important tasks first.
- Works now but causes confusion later: second order thinking = short-term success may create long-term disorder.

### Murabbi Notes

Keep the discussion short and sharp. The lesson should train shabab to think with tools, not just opinions.
