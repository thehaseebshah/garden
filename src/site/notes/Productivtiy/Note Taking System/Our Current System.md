- For focused note taking use the eInk tablet. We also use it for journalling.
- For certain projects/topics take printouts, this aids memory, allows recall techniques.
- Analog A6 Notebook for Inbox, Tasks and Planning, but the main storage remains on Obsidian.
- Obsidian on Laptop for less focused idea notes, ref, notes that need indexing
- Google Docs for collaboration.
- Calendar stays on phone for reminders.
- Some notes can be as PDFs and Office docs in Documents folder on our computer.
- For long project notes use Obsidian, or where the project will be done.
