- Have a Digital Storage System or Task Manager
	- It should have these 4 folders:
		- Today
		- This Week
			- This should not contain more than 25 tasks
		- Someday
	- We have simple label lists for categories
	- For Large Projects have a folder in your Google Drive or a Note in your Note Taking app; in your weekly plan you can assign tasks to yourself to work on your projects.
	- For Small Projects you can have tasks with sub-tasks.
- If you want to go pro you can do this analogue

- Additional Info:
	- Inspired from Getting Things Done,  [[Ideas/Zen to Done\|Zen to Done]] and Cal Newport
